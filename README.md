# ♍ Virgo

>A beginner-friendly Obsidian vault built for people who want to spend less time organizing and more time thinking.

## Why Virgo?

When I first started using Obsidian, I tried several popular vaults. While they were powerful, many felt overwhelming for someone just getting started. They came with complex workflows, countless folders, and systems that often felt harder to learn than the app itself.

So I built **Virgo.** 

Virgo combines ideas inspired by different community vaults while keeping the experience clean, simple, and approachable. It isn't designed to be the most feature-rich vault available. Instead, its goal is to provide a solid foundation that beginners can understand immediately and experienced users can easily customize.

If you're looking for a vault that gets out of your way and lets you focus on your notes, Virgo is for you.

## Features

- Clean folder structure
- Beginner-friendly workflow
- Reusable note and project templates
- Organized project management
- Easy to customize and expand
- Minimal setup required

## Folder Structure

```text
├──_Index/

02-Notes/
├──_Index/

Archive/

Linked-Files/
```

## Philosophy

Virgo follows one simple idea:

>Your note-taking system should support your thinking—not become another thing you have to manage.

Start simple. Add complexity only when you actually need it.

## Getting Started

1. Download or clone this repository.
2. Open the folder as an Obsidian vault.
3. Start writing.

No complicated setup required.

## Why the name "Virgo"?

The name comes from my zodiac sign. This vault began as a personal workspace designed around the way I think and organize information. Over time, it became structured enough that I wanted to share it with others who might be looking for a simpler way to use Obsidian.
