# 🎯 Projects

A project has a goal and an end date. If it doesn't, it's probably a [[02-Notes/_Index|Note]] or an ongoing area instead.

## How this folder works
- One folder per project, add subfolders as needed such as Logs, and related notes.
- One file per project, created from the [[00-Inbox/Templates/Project Template|Project Template]].
- When a project finishes or dies, move its file to `Archive` and update the status below.
