# 📥 Inbox

Capture everything here first. Don't sort on the way in — sort on the way out.

## How this folder works
- New thought, link, idea, task, or half-formed note? Drop it here, fast, unfiltered.
- Once a week, process the inbox to zero:
  - Belongs to a project → move to `01-Projects`
  - Standalone idea or reference → move to `02-Notes`
  - Done, irrelevant, or stale → move to `Archive`
- If you can't tell what it is yet, leave it — that's what this folder is for.
- Add folders if you think they're needed — totally optional, don't overthink it.
